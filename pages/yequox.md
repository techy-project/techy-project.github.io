# Yequox
**Yequox** (pronounced Ye-qoks) is a Linux distribution based on Ubuntu with the KDE Plasma desktop. **Yequox** is also known for its handbook, where you can build your Linux distribution manually in both UEFI and BIOS.
# OpenYequox
**OpenYequox** (a.k.a. **Open Palm Environment Needing Yequox**) is a lightweight version of Yequox. It uses the Enlightenment window manager instead of the KDE Plasma desktop. It has no Plymouth, LibreOffice and more but except for Firefox.
