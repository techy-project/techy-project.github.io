# bmXos
**bmXos** (a.k.a. **beam X11 operating system**) is a Linux distro based on Linux Mint released on December 10, 2010. The distro now has an official release after 3 years after its distro release (December 10, 2010).
# Naming
The **bmXos** name originally comes from these 2 Linux distros made from Pinguy Builder, like **BM**_nux from [nur cahyaeni](https://www.youtube.com/@nurcahyaeni7574), and **Xos**, **X os**, Linux **X os** or Linux**Xos** from [Richard Wiryadinata](https://www.youtube.com/@richardwiryadinata632).
The **bmXos** name also have different names like, **beamX11os**, **beamXos**, **BMX11 Linux**, **BMnuX OS** and **beamLinuX**.

## Alpha Release
The **bmXos 2.10.2 Alpha** release is an alpha release, which is released on December 12, 2012 and based on Linux Mint 12.
Discontinued after the official **bmXos 2.11.4** release, and also never released on the public.
## Never Released
The **bmXos 2.9.1 Pre-Alpha** release is a never released pre-alpha, released on November 9, 2010 and based on Linux Mint 9.
Discontinued after the **2.10.2** alpha release.
