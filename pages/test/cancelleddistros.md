# Cancelled distros
## Cubuntu
I was working on Cubuntu back in 2022, using the discontinued DistroShare Ubuntu Imager, then got it. The first and only version was 12.0 and I never released 12.1, 12.2 or 13.0. It is based on Ubuntu and uses the GNOME desktop, slightly modified, and I thought it would be a official Ubuntu flavor. It is discontinued in August 2023 and cancelled in September 2024. It is gonna be replaced by tuBuntu 13.0 since Cubuntu already exists with the version 16.04.
